# KantoModels - Issue Tracker

This repository is used only for bug reports, compatibility issues, and support requests related to the KantoModels Minecraft mod.

The source code is currently private and is not distributed publicly.

## Before opening an issue, please include:

- Minecraft version
- Fabric Loader version
- Fabric API version
- KantoModels version
- Framework version
- GeckoLib version
- Crash report or latest.log, if applicable
- Steps to reproduce the issue

## Official pages

Downloads, changelogs, and mod information are available on the official CurseForge and Modrinth pages.
